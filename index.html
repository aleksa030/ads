Usluge

Nudimo Vam širok spektar naših usluga. Radimo sve vrste izrada web site-ova kao i održavanje istih po najpovoljnijim cenama. Najsavremenije dizajniran web site je sigurno ono što je potrebno svakom ko se bavi bilo kakvim biznisom. Takođe, radimo i prepravke delova ili prenamenu celih web site-ova Uverite se u naš kvalitet, vaš Aleksa Design studio!

HTML

HTML jezik je osnova svake web stranice, uz pomoć koga se prezentuje izgled i sadržaj iste. Omogućava formatiranje stranice, obradu slika, teksta i još mnogo toga.

CSS

CSS je skraćenica od Cascading Style Sheets. Radi se o stilskom jeziku, koji se koristi za opis prezentacije dokumenta napisanog pomoću markup (HTML) jezika. Bez css stilskog jezika ne bi bilo moguće adekvatno stilizovati web stranicu.

JAVASCRIPT

Javascript je jedna od tri vodeće tehnologije za definisanje sadržaja na web-u. Većina web-sajtova koristi Javascript. Kombinovan sa HTML jezikom i CSS-om Javascript čini DHTML (Dynamic HTML).

JQUERY

JQuery je višeplatformska Javascript biblioteka dizajnirana da pojednostavi skriptovanje sa korisničke strane jezika HTML. Najpopularnija je Javascript biblioteka u upotrebi danas koju koristi 65% najposećenijih sajtova na webu.
